# FeedEater
シンプルで高速なWindowsデスクトップ向けRSSリーダーです。

## 主な機能
* **複数フィードの登録**: 任意のRSS/AtomフィードのURLから記事を追加できます。
* **未読管理**: 未読の記事が一目で分かり、選択すると自動的に既読になります。また、記事やフィードを右クリックして「未読にする」ことも可能です。
* **キーボード・ナビゲーション**: マウス不要でキーボード (`j` / `k` または上下矢印キー) のみで素早く一覧を閲覧できます。さらに、`Shift + j` / `Shift + k` を使うと次の未読記事、前の未読記事へ一気にジャンプできます。
* **画像・HTMLのインライン表示**: 本文中の画像を表示しつつ、レイアウトの崩れを防ぐシンプルな３ペインUIを採用しています。
* **OS標準ブラウザ連携**: 外部リンクや「続きを読む」のURLは、アプリ内ではなく普段お使いの規定ブラウザ（ChromeやEdgeなど）で安全に開かれます。
* **フォルダ管理**: フィードをフォルダに分けて整理できます。フォルダは折りたたみが可能で、多くのフィードを効率的に管理できます。

## 使い方
1. アプリを起動すると、3ペイン（左から「フィード一覧」「記事一覧」「本文」）の画面が開きます。
2. 左上の `+` ボタンをクリックし、購読したいウェブサイトのRSS URL（例: `https://zenn.dev/feed`）を入力して追加します。
3. フォルダを作成するには、左上のフォルダアイコンをクリックします。
4. フォルダにフィードを移動するには、フィードを右クリックして「フォルダに移動」から目的のフォルダを選択します。フォルダから出す場合も同様に右クリックメニューから操作できます。
5. 取得された記事一覧から読みたいものを選択すると、右側のペインに本文が表示されます。
6. 記事を読み終えたら `j` キーで次の記事へ進むことができます。さらに `Shift + j` で次の未読記事へジャンプできます（戻る場合は `Shift + k`）。
7. 記事を間違えて既読にしてしまった場合は、記事一覧で右クリックして「未読にする」を選択できます。フィード全体を未読にする場合は、左ペインのフィードを右クリックします。
8. 登録したフィードを削除したい場合は、左ペインのフィード一覧から該当のフィードを右クリックし、「削除」を選択します。
9. フォルダを削除したい場合は、フォルダを右クリックして「削除」を選択します。フォルダを削除しても中のフィードは削除されず、未分類の状態に戻ります。

## リリース履歴

### v0.8.0 (最新)
* **フィード・記事を未読に戻す機能の完成**: 
  - フィード一覧、記事一覧、および記事詳細（本文ペイン）上での右クリックコンテキストメニューから「未読にする」を実行できるようにしました。
  - 記事を未読に戻した後、別の記事に切り替える（フォーカスを外す）まではその記事を未読のまま維持するガード処理を導入しました（「あとで読む」目印としての機能）。
  - Electronのコンテキストメニュー表示時に、メニューが閉じる際のイベントとクリックハンドラーが非同期で競合してキャンセルされる不具合を修正しました。
* **コード品質の改善**:
  - テストコード内に残っていたTypeScriptのモック型キャスト警告を `vi.mocked` にリファクタリングし、静的検証（Lint）の警告を完全に解消しました。

### v0.7.0
* **フォルダ管理機能の追加**: 左ペインにフォルダを作成し、任意のフィードをフォルダへ移動して整理できるようになりました。
* **セキュリティの強化**: カスタムプロトコル (`app://`) の導入や、外部ブラウザ連携時・フィード取得時のURL妥当性検証の追加。
* **テストの強化**: テストカバレッジの向上。
